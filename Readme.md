# AutoServe 2.0
AutoServe 2.0 is a web application that allows users to easily create and manage their own web servers. It provides a user-friendly interface for configuring server settings, managing files, and monitoring server performance.
## Features
- Easy server setup and configuration
- File management system- Performance monitoring tools
- User authentication and access control
- Real-time server status updates
## Installation
To install AutoServe 2.0, follow these steps:
1. Clone the repository: `git clone https://github.com/adii4040/AutoServe.git`
2. Navigate to the project directory: `cd AutoServe`    
3. Install dependencies for the client: `cd Client && npm install`
4. Install dependencies for the server: `cd ../Server && npm install`
5. Start the server: `npm start`
6. Start the client: `cd ../Client && npm start`
7. Open your browser and navigate to `http://localhost:3000` to access the application.
